# Számla fordító webes felület

Egyszerű, standard library alapú Python webalkalmazás, ami PDF formátumú külföldi számlából automatikusan elkészít egy magyar nyelvű `.docx` dokumentumot.

## Indítás

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Ezután: `http://localhost:5000`
